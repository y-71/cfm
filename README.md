# Typelate 🖖👽🖖

Typelate is an ultra fast typescript template running with `sucrase` and `rollup` 🚀

Typelate also supports jest tests.🧪

It's optimized for small typescript projects. 🤏

No server needed.🖖

# Compiling Typelate ⚙️
```yarn```
# running tests 🧪
```yarn test```
# running the `index.ts` file:
```yarn start```
